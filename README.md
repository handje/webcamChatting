# webcam_chatting
Zoom clone using NodeJS, WebRTC and Websockets

1. room 이름을 통해 채팅방 생성 및 참여
2. 카메라, 오디오 요청
3. 자신의 화면과 음소거버튼, 화면전환 버튼, 카메라 목록을 보여줌
4. 상대방이 입장하면 자신의 화면 옆에 상대방의 화면 출력
5. 텍스트 채팅 가능
